# Product CRUD Plugin

WordPress plugin that registers a custom "Product" post type with REST API support.

## Features

- Custom post type: `product`
- REST API endpoint: `/wp-json/wp/v2/products`
- Custom fields: price, SKU, stock
- Full CRUD support via REST API

## Installation

1. Copy this folder to `wp-content/plugins/`
2. Activate the plugin in WordPress admin
3. Access products via REST API at `/wp-json/wp/v2/products`
